# Robo School — Education Landing Page 🤖

Современный адаптивный лендинг для школы робототехники. Проект выполнен на чистом стеке (HTML/CSS) с упором на мобильную адаптацию и аккуратную типографику.

## 🛠 Технологический стек

- **HTML5:** Семантическая верстка.
- **CSS3:** Flexbox-сетки, реализация мобильного адаптива и кастомная стилизация.
- **Fonts:** Интеграция шрифтов через Google Fonts.
- **Icons:** [Font Awesome 6](https://fontawesome.com/) — векторные иконки.

---

## 🚀 Как запустить проект

1. Откройте папку проекта в **VS Code**.
2. Запустите расширение **Live Server**.
3. Проект откроется автоматически по адресу `http://127.0.0.1:5500`.

---

## 📱 Особенности реализации

- **Responsive Design:** Полная адаптация под смартфоны и планшеты с использованием Media Queries.
- **Clean UI:** Простой и понятный интерфейс, ориентированный на образовательную тематику.
- **Mobile First:** Акцент на удобстве использования сайта на маленьких экранах.
- **Zero JS Dependencies:** Проект работает быстро благодаря отсутствию тяжелых JavaScript-библиотек.
